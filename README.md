# Features

- 没有任何依赖
- 支持 `AMD` & `CommonJS`
- 轻量级（10kb）


>单元测试:http://hawx1993.github.io/judge/test/

## Quick Start

```js
$ npm install  
$ gulp compress  
```

>使用`npm`安装`judgejs`：

```js
$ npm install judgejs
```

>使用`bower` 安装`judgejs`

```js
$ bower install judgejs  
```
